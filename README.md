# kawaapp-privacy

iOS アプリ「**河川点検士 試験対策**」（リポジトリ名: `kasen_inspector`、Bundle ID: `com.kawaapp.kasen_inspector`）の
プライバシーポリシーおよび出典・免責事項を公開するための GitHub Pages リポジトリ。

> **NOTE**: 命名が `kawaapp-privacy` となっているのは初期の名残で、実体は河川点検士アプリ専用。
> 将来「`kasen-legal` を新規作成しよう」とならないよう、ここに明記しておく。
> 他アプリの法務ページは別リポジトリ:
> - `kyoryo-legal` … 道路橋点検士 (`kyoryo_inspector`)
> - `drone-ittou-legal` … 一等学科試験対策 (`drone_pilot`)
> - `drone-nittou-legal` … 二等学科試験対策 (`drone_pilot_nittou`)
> - `rikutoku-legal` … 2陸特・3陸特 合格対策 (`rikutoku_quiz`)

## 公開URL

- プライバシーポリシー: https://takurofukuma.github.io/kawaapp-privacy/
- 出典・免責事項: https://takurofukuma.github.io/kawaapp-privacy/sources.html

## 構成

| ファイル | 用途 |
|---|---|
| `index.html` | プライバシーポリシー（App Store Connect の Privacy Policy URL） |
| `sources.html` | 参考にした資料の一覧と免責事項（App Store Connect の Support URL） |

## 更新方法

ローカルで編集 → `git push origin main` で GitHub Pages に即反映（数十秒）。
